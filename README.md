# AssetTracker (Telegram-bot)
Телеграм бот-помощник в управлении с активами. Добавляйте, обновляйте и удаляйте активы, смотрите статистику по вашему портфелю активов и торгуйте с нейросетью. Gemini-2.5-flash дает индивидуальные советы по вашему портфелю исходя из текущей ситуации на рынке.
